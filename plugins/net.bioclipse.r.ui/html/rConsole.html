R Console

R Console is the scripting console for the statistical programming language R, which gives the user access to data and to an interactive command-oriented R environment.

Starting the R Console

The R Console can be displayed through Window > Show View > Other... > Scripting > JavaScript Console".

Another way to display the R console is by opening a R script in the R editor and sourcing it, or running R code from the editor (read more at the R editor page).
